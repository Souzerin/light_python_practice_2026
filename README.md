# Практика по Python, 2026
Консольный индексатор файлов на Python с хранением данных в SQLite.

## Возможности

- Рекурсивное сканирование папок
- Фильтрация файлов по расширению и имени
- Поиск дубликатов по хэшу содержимого (SHA256)
- Сравнение папок с резервными копиями
- Хранение индекса и истории проверок в SQLite

## Запуск программы

Запуск происходит из консоли с помощью команды:

```
py src/main.py /путь/к/папке
```

Нужно находиться в корневой директории проекта.

## Структура проекта

```
project/
├── src/
│   ├── main.py          # Точка входа
│   ├── database.py      # Работа с SQLite
│   ├── scanner.py       # Сканирование ФС
│   ├── indexer.py       # Индексация файлов
│   ├── filters.py       # Фильтры
│   ├── hasher.py        # Хэширование
│   ├── duplicates.py    # Поиск дубликатов
│   ├── backup.py        # Сравнение с бэкапом
│   └── reporter.py      # Отчёты
├── data/                # База данных
│   └── app.db
├── check_db.py          # Проверка БД
└── README.md
```
