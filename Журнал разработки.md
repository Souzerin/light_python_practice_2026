## 24.06.2026

**Этап:** Исправление ошибок

**Сделано:**
- Исправлена ошибка `UNIQUE constraint failed: files.relative_path` в `indexer.py`
- Причина: при повторном сканировании не учитывались файлы с `is_present = 0`
- Решение: запрос всех файлов из БД, а не только присутствующих
- Функция `reset_index()` для полной очистки индекса
